Project Background /n
Risk Averse, LLC is looking for a complete analysis of a specific natural hazard across six states in the same region. This project will provide the analysis of hurricane patterns in Alabama, Florida, Georgia, Louisiana, Mississippi, and Texas by utilizing python code to generate plots that are clear and descriptive of the data that they represent. The natural hazard and census data featured in the following graphs is sourced from NRI Census Tract and SVI datasets for the given states.
Scope of Work /n
The goal of the project is to analyze the risk of a specific natural hazard, hurricanes, among 6 different states, Texas, Louisiana, Mississippi, Alabama, Georgia, and Florida, and the impact of the hazard. The analysis is to consist of a comparison of how the risk of hurricanes vary across these 6 different states, a report of any trends that occurred in the data, what these trends may indicate about the region, and a discussion of how this natural hazard has impacted the communities of these areas.
Tasks to accomplish the project goal include extracting and organizing the SVI data, preliminary research, examining and evaluating the risk, plotting data, and reporting the data.
Project deliverables include the following: raw data files used in the python code file, the coding file used to generate the solution, scope of work, Gantt chart, engineering timesheet, annotated code document, a link to the Github repository with all project files, and the written report stating the results and findings of the project.
